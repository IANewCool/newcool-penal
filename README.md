# NewCool Penal - Derecho Penal Chile

Portal educativo sobre derecho penal y procedimientos en Chile.

## Demo

[https://penal.newcool.io](https://penal.newcool.io)

## Caracteristicas

- Calculadora de penas
- Beneficios intrapenitenciarios
- Guia de procedimiento penal
- Tipos de delitos
- Derechos del imputado

## Tecnologias

- Next.js 15
- React 19
- Tailwind CSS 4
- TypeScript

## Desarrollo

```bash
npm install
npm run dev
```

## Parte del Ecosistema NewCool

Este modulo es parte de [NewCooltura Informada](https://informada.newcool.io) - Portal de instituciones publicas de Chile.
